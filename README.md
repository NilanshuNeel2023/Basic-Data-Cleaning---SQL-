# Data Cleaning Project Using SQL
## Project Overview

This project focuses on performing real-world data cleaning operations using SQL to transform raw and inconsistent data into a clean, accurate, and analysis-ready dataset.

Data cleaning is one of the most critical steps in the data analytics process. In this project, multiple data quality issues such as:

- Duplicate records
- Missing values
- Invalid email formats
- Incorrect spellings
- Extra spaces
- Inconsistent date formats
- Negative values
- Incorrect data types

were identified and resolved using SQL queries.

This project demonstrates practical expertise in:

- SQL Data Cleaning
- Data Transformation
- Data Validation
- Data Standardization
- Data Quality Management
- Real-World Problem Solving

## Business Problem

Raw business data often contains inconsistencies and errors that can negatively impact reporting, dashboarding, and decision-making.
This project addresses common real-world data issues by:

- Identifying inaccurate records
- Standardizing data formats
- Removing duplicate entries
- Correcting invalid values
- Preparing the dataset for analysis and reporting

The final cleaned dataset becomes reliable, consistent, and ready for business intelligence analysis.
